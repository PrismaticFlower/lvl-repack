Built with .NET Framework 4.6 (oldest sdk you can still easily get)
download .net framework sdk (4.6) from:
   https://dotnet.microsoft.com/en-us/download/visual-studio-sdks

.NET Framework comes with Windows by default and is forward compatible 
(projects compiled with older versions will run on newer runtimes)

Here is the list of which .Net Frameworks come with which versions of Windows:
=========================================
Windows 7:    .NET Framework 3.5
Windows 8:    .NET Framework 4.5
Windows 10:   .NET Framework 4.6-4.8
Windows 11:   .NET Framework 4.8
=========================================
